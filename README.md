## Trabajo_intro

## INSTALL
```bash
python3 -m venv venv
source venv/bin/activate
pip install Flask
pip install flask_cors
pip install flask_sqlalchemy
pip install psycopg2-binary

```
## RUN
```bash
source venv/bin/activate
cd backend
flask run --debug
```
## Abro otra terminal
```bash
cd frontend
python3 -m http.server
```
